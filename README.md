# React-Webpack-TypeScript-Babel

> A starter template for projects using React, Webpack, TypeScript and Babel.

## Starting the Dev Server

```sh
npm start
```

## Building the Bundle

```sh
npm run build
```

## Type-checking the Project

```sh
npm run typecheck
```

And to run in `--watch` mode:

```sh
npm run typecheck:watch
```
